# File Display

## Case

Assume that a file containing a series of integers is named numbers.dat and exists on the computerâ€™s disk. Design a program that displays all of the numbers in the file

<hr>

## Pseudocode

```
Declare Integer number
Declare InputFile listNumbers
Open listNumbers "numbers.dat"

While NOT eof(listNumber) Then
    Read listNumbers numbers
    Output numbers
End While

Close listNumbers
```
